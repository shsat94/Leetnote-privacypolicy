👋 Introduction

Thank you for using LeetNotes.

LeetNotes is a Chrome extension that helps you save and organize your accepted LeetCode solutions by uploading them to your GitHub repository after your confirmation.

Your privacy is important to us. This Privacy Policy explains what information the extension accesses, how it is used, and what is never collected.

🔍 Information We Access

LeetNotes accesses only the information necessary to provide its core functionality, including:

  • Accepted LeetCode solution code
  • Problem title and number
  • Programming language
  • Runtime and memory statistics
  • Submission timestamp
  • GitHub Personal Access Token (PAT)
  • Repository name and extension preferences

⚙️ How Your Information Is Used

The accessed information is used solely to:

  • Upload your accepted LeetCode solutions to your GitHub repository.
  • Organize your solutions according to your selected folder structure.
  • Save your extension settings locally.

LeetNotes does not use your information for advertising, analytics, profiling, tracking, or marketing.

No ads No analytics No tracking

💾 Data Storage

All settings, including your GitHub Personal Access Token, are stored locally using Chrome Extension Storage.

LeetNotes does not maintain any servers and does not collect or store your personal information remotely.

🔑 GitHub Authentication

Version 1 of LeetNotes uses a GitHub Personal Access Token (PAT) supplied by you.

The token is used exclusively to communicate with GitHub's official API for uploading files into repositories you own or select.

Your GitHub token is never transmitted to the developer and is never stored on external servers.

🔗 Third-Party Services

LeetNotes communicates only with:

  • LeetCode (to detect accepted submissions)
  • GitHub API (to upload your solutions)

No other third-party services are used.

🚫 Data Sharing

LeetNotes does not:

  • Sell your data
  • Share your data
  • Rent your data
  • Transfer your data to advertisers
  • Use your data for analytics
  • Track your browsing activity

🛡️ Permissions

LeetNotes requests only the permissions required to:

  • Access LeetCode pages
  • Store extension settings locally
  • Communicate with GitHub APIs

No unnecessary permissions are requested.

🎛️ Your Control

You remain in complete control of your data.

You may:

  • Remove your GitHub token at any time
  • Disable the extension
  • Uninstall the extension
  • Delete uploaded repositories directly from GitHub

📝 Changes to This Privacy Policy

This Privacy Policy may be updated as LeetNotes evolves. Any significant changes will be reflected by updating the "Last Updated" date.

✅ Summary

LeetNotes is designed with privacy as a priority.

Your LeetCode solutions remain yours. Your GitHub repository remains yours. Your GitHub token stays on your device.

LeetNotes does not collect analytics, track users, or send personal information to the developer.
